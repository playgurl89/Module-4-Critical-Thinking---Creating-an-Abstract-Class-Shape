# Shape Array Project (CSC372)

This project was created for my CSC372 class to practice object-oriented programming in Java, specifically abstraction, inheritance, and polymorphism.

I built an abstract Shape class and then extended it with three subclasses: Sphere, Cylinder, and Cone. Each class has its own implementation of surface area and volume calculations.

The program also includes a driver class (ShapeArray) that creates one of each shape, stores them in an array of Shape objects, and then loops through the array to print each shape using polymorphism.

## What I built
- Abstract Shape class with required methods
- Sphere, Cylinder, and Cone subclasses
- Surface area and volume calculations for each shape
- ShapeArray driver class using a Shape array
- UML diagram showing class relationships

## What I practiced
- Abstract classes in Java
- Method overriding
- Inheritance and polymorphism
- Using arrays of objects with a superclass type
- Writing and organizing multi-file Java projects
- Git version control with multiple commits

## How to run it
Compile all `.java` files and run `ShapeArray.java`.

Example:
javac *.java
java ShapeArray
